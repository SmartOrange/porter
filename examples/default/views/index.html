Oceanify Exmaple

This is an example koa app with oceanify setup, with its source available at github.Oceanify is a consolidated solution for browser module authoring. It provides

  • a middleware for browser module transformation,
  • a loader for browser module loading,
  • and a compiler for browser module compilation to make the app production ready.
for brevity concern, we'll refer to browser module as component in the rest of this page.

Setup

In app.js, setup the middleware.

In html, use the loader to import the entry component.

In file components/home.js, require all the way.

In file components/stylesheets/app.css, there's @import.
